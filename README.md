# kit-gpg-portable windows 10/11
KIT “portable” (Windows) per **cifratura/decifratura con GPG** e gestione operativa di scambio file riservati, pensato per scenari in cui non puoi aprire un servizio ad hoc, vuoi restare su strumenti standard e verificabili in grado di rendere ripetibile e controllabile uno scambio di file cifrati 
Il kit funziona anche se lasciato su chiavetta
